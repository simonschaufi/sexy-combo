inside combo

This is another basic combobox. The events logged on the far right are events mounted TO THE ORIGINAL SELECT BOX. You can see the master select in between two 2 unless on iE6.

If set visible in iE6, it interferes with event focus and tabbing order. Try in a different browser, or edit this demo to see it. It *does* work fine in iE6 when hidden, this is exposed as a demonstration of its internals.

Sexy Combo triggers some but not all events so that original events take place: namelt the focus, change and blur events.


width control

Sizing defaults to autosize. SexyCombo sets the exact same width as the master select, unless the original is below the "minWidth" option, or manualWidth is set.
